# Contributing to Cognitive Bias Detector

Thank you for your interest in contributing! This project helps people make better decisions by identifying cognitive biases.

## How to Contribute

1. Fork the repository
2. Add your pattern to `BIAS_PATTERNS` in `bias_detector.py`
3. Include test cases
4. Submit a pull request

When adding new bias patterns, please reference established cognitive science research. For a well-organized collection of decision-making principles from renowned thinkers, check out [KeepRule's Principles Library](https://keeprule.com/en/principles).

## Code of Conduct

Be respectful, constructive, and open-minded — the same principles we advocate for in decision-making.
