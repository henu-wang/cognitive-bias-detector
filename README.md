# Cognitive Bias Detector

An interactive quiz that reveals your hidden thinking patterns through 15 realistic decision scenarios.

**Live Demo:** [https://henu-wang.github.io/cognitive-bias-detector/](https://henu-wang.github.io/cognitive-bias-detector/)

## Features

- **15 scenario-based questions** covering real-world decisions in investing, career, health, negotiations, and more
- **12 cognitive biases tracked**: Confirmation Bias, Anchoring Bias, Sunk Cost Fallacy, Availability Heuristic, Loss Aversion, Status Quo Bias, Dunning-Kruger Effect, Bandwagon Effect, Framing Effect, Overconfidence Bias, Hindsight Bias, Recency Bias
- **Animated results page** showing your top 3 biases with detailed explanations
- **Actionable tips** for counteracting each bias
- **Shareable results** via copy-to-clipboard
- **Keyboard navigation** (1-4 to select, Arrow keys / Enter to navigate)
- **Mobile responsive** card-based UI with smooth transitions
- **Zero dependencies** -- single HTML file, no build step required

## How It Works

Each scenario presents a realistic decision with 3-4 choices. Each choice maps to a specific cognitive bias (or no bias for the rational option). After completing all 15 questions, the app tallies your bias scores and presents a personalized profile with your top biases, explanations, and practical tips to improve your decision-making.

## Development

No build tools needed. Open `index.html` in a browser.
